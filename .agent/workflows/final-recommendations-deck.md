## Purpose
Deliver BCG's final recommendations to the client with full analytical backing, quantified value, and an actionable implementation roadmap.

## Contents
1. Executive Summary (1 slide — the answer up front)
2. Situation and Problem (1-2 slides — aligned context)
3. Key Findings by Workstream (structured findings, 4-6 slides each)
4. Integrated Recommendations (3-5 headline recommendations with value estimates)
5. Implementation Roadmap (phased action plan, 12-24 months)
6. Risks and Mitigations
7. Appendices (supporting analytics, benchmarks, data exhibits)

## Team Structure: Full
- **analyst** leads and coordinates:
  - **analyst** (×N) — Per-workstream analytical sections
  - **deck-builder** — Full slide production
  - **program-manager** — Implementation roadmap
- analyst runs internal review before partner escalation

## Review Panel (independent, parallel)
- **partner-strategy** — Strategic alignment and commercial credibility
- **partner-analytics** — Analytical rigor and data accuracy
- **principal-delivery** — Implementation feasibility

## Output Format
Full slide outline in markdown with all content populated.
Appendix index included.

## Quality Gates
- Executive summary is a standalone answer (readable without rest of deck)
- All recommendations are specific: who, what, when, expected outcome
- All value estimates have explicit methodology
- Implementation roadmap has owners, not just teams
- No finding in the appendix contradicts the main deck
